Operating Introduction

Keyboard function

  • key 1:Release first portable skill.
  • key 2:Release second protable skill.
  • key R, T:Release ultimate skill 1, ultimate skill 2.
  • key W, A, S, D: Control the role move.
  • key O: Open/Close full screen.
  • key F: Place the mark flag.

Mouse function

  • Move mouse: Rotate the vision.
  • Click: Normal attack.

Portable Skill Introduction

Accelerate

Release skills to gain acceleration for 5 seconds. CD: 20s

Destroy

Shot magic ball to demolish the wall. CD: 47s

Sniper

Get additional attack skill, this attack is ranged attack. CD: 30s

Defend

Every miunte you'll get "gold body" to block an attack no matter imprison, sniper or normal attack. This skill is a passive skill, it will auto trigger when you get attack.

Detect

As the skill name, when you release this skill, you'll scanning a fixed range if have opponents or not. (The red point are opponents, blue point is you.) CD: 40s

Role Introduction


class 0

Freedom Wizard

  Just a wizard want to get free, but he doesn't have any special skills. The only way to help him get special skills is own our NFT skills so that Freedom Wizard will be happy.

  This wizard is the basic wizard if you doesn't have our NFT roles.

class 1

All the class 1 roles can choose two portable skills, and they all have their special ability.

space wizard

  Space wizard has special space ability, when he release his ultimate skill, the coin's position will be forced to refresh to another romdom point. CD: 62s

hidden wizard

  Hidden wizard can confuse enemy's vision let them can't see him, but he can't hidden his smoke under the robe. Notice that during invisibility, hidden wizard still can be attack. CD: 35s

imprison wizard

  Imprison wizard like "bind play" with everyone, he can't control himself to bind other wizards. Be careful his imprison magic, if you are hit, you can move and attack for 4 seconds. CD: 30s

class 2

All the class 2 roles can choose two portable skills, and they all have their special ability.

You need to collect two specific mystery fragments to synthesis one class 2 role.

finagle wizard

  Finagle wizard was a habitual fraudster, so he has the ability to make a lot of fake money.

  His special ability is that he can make some fake money around the real money, and random position at the map. If another player touch the fake money, it'll be returned to respawn point. CD: 65s
