Game Introduction

Panning Holic

  The game is developed by three university students, you can pay some entry fee to play this game through ThunderCore Mainnet, all the transaction we will draw 3% as our benefit. If you want to support our team, you can transfer some ETH or TT to this address: 0xa8aD68EEFe1dEcdfb0353963b38c502018f08546, we will be very grateful to you🎉🎉🎉.


  When the game start, you will see the button "login with MetaMask", we will do something: connect to our website, and check your MetaMask network whether checked ThunderCore Mainnet or not. If your MetaMask network doesn't have ThunderCore Mainnet, our game will add and switch to ThunderCore Mainnet automatically.


  This game is a four-player game, you will gather in one room to play this horror game, use your skills and roles which you have to scramble the coin twice, you will win the game. Make sure you won't be intimidated by other players🥴.


  Each player can attack with each other, if you get hurt, you will return to respawn point.


  To avoid you get lost, you can use the mark flag, notice that a map only exist three flags at the same time, when you put fourth flag, the first flag will be remove.


  If you are the winner, you will win the entry fee paid by the remaining three players. Hope you have fun!🎊

Operating Introduction

Keyboard function

  • key 1:Release first portable skill.
  • key 2:Release second protable skill.
  • key R, T:Release ultimate skill 1, ultimate skill 2.
  • key W, A, S, D: Control the role move.
  • key O: Open/Close full screen.
  • key F: Place the mark flag.

Mouse function

  • Move mouse: Rotate the vision.
  • Click: Normal attack.

Portable Skill Introduction

Accelerate

Release skills to gain acceleration for 5 seconds. CD: 20s

Destroy

Shot magic ball to demolish the wall. CD: 47s

Sniper

Get additional attack skill, this attack is ranged attack. CD: 30s

Defend

Every miunte you'll get "gold body" to block an attack no matter imprison, sniper or normal attack. This skill is a passive skill, it will auto trigger when you get attack.

Detect

As the skill name, when you release this skill, you'll scanning a fixed range if have opponents or not. (The red point are opponents, blue point is you.) CD: 40s

Role Introduction


class 0

Freedom Wizard

  Just a wizard want to get free, but he doesn't have any special skills. The only way to help him get special skills is own our NFT skills so that Freedom Wizard will be happy.

  This wizard is the basic wizard if you doesn't have our NFT roles.

class 1

All the class 1 roles can choose two portable skills, and they all have their special ability.

space wizard

  Space wizard has special space ability, when he release his ultimate skill, the coin's position will be forced to refresh to another romdom point. CD: 62s

hidden wizard

  Hidden wizard can confuse enemy's vision let them can't see him, but he can't hidden his smoke under the robe. Notice that during invisibility, hidden wizard still can be attack. CD: 35s

imprison wizard

  Imprison wizard like "bind play" with everyone, he can't control himself to bind other wizards. Be careful his imprison magic, if you are hit, you can move and attack for 4 seconds. CD: 30s

class 2

All the class 2 roles can choose two portable skills, and they all have their special ability.

You need to collect two specific mystery fragments to synthesis one class 2 role.

finagle wizard

  Finagle wizard was a habitual fraudster, so he has the ability to make a lot of fake money.

  His special ability is that he can make some fake money around the real money, and random position at the map. If another player touch the fake money, it'll be returned to respawn point. CD: 65s

lost wizard

  迷航者生前是一名船員，因為他的方向感太差導致他永遠無法晉升為船長成為他這輩子的遺憾。

  他的終極技能為，釋放自己的腦電波干擾其餘法師的羅盤使他們的羅盤失效10秒，並清除地圖上其他玩家的標記旗幟，讓所有人都和他一樣迷失方向是他快樂的來源。 CD：37s

傳送者

  傳送者是一名科學家，主要研究時空關係並造出了自己的時光機，是的，他還沒有死亡，但在最後一次失敗的實驗中，他駕駛自己製作的時光機卻意外來到這個世界。

  他的終極技能為，他將利用自己的時光機傳送其他法師至地圖上的某個位置，不過由於時光機的製作不良，他沒辦法控制其他法師會被傳送到哪個位置，也許會來到金幣旁邊。。？ CD： 60s

class 3

所有三階角色都可以攜帶兩個可攜帶技能，他們的特殊能力幾乎可以左右整場比賽的勝利，三階角色的普通攻擊一旦成功擊殺目標會造成目標減少一顆已獲得的金幣。

你需要獲得超稀有碎片以及兩個特定的神秘碎片方可合成出一隻三階角色。

金牌刺客

  「我們是龍的傳人......」隨著一遍遍的口號響起，人手一枝的火炬逐漸將這已經足夠遼闊的廣場塞得水洩不通，這個夜晚的天空除了漫天的繁星閃爍，也被地面的火光照的熠熠生輝，雖然成為金牌刺客的人都是萬里挑一，但在龐大的人口基數下能夠有如此盛景也是不足為奇。他們全身火紅，仔細一看，全是千錘百煉的痕跡，沒人知道在成為金牌刺客之前他們受了多少常人無法忍受的痛苦，但所有人都知道，當你見到他們的那一刻，距離死亡已經不遠了。

  金牌刺客同時擁有束縛技能以及隱身技能，同時隱身期間他們的行走速度比平時更快，若在隱身期間使用普通攻擊擊殺目標，將會刷新隱身的冷卻時間。

絕對防護

  「嘩......」火熱的岩漿從頭頂淋下，從頭到腳的每一吋肌膚都感受到火熱的痛楚，這是成為絕對防護的最後一道關卡，只有承受住這樣的傷害，才能成為真正的絕對防護，但經過這次岩漿的洗禮後，原本透過訓練而泛出的金光也會消失，取而代之的是全身通紅，所有想靠近他們並攻擊的人都將自食惡果。大部分的絕對防護都是預感到了自己將要成為金牌刺客的攻擊目標，為了與金牌刺客抗衡只好練就對抗他們的能力，即便訓練的過程也很可能喪命。

  絕對防護擁有自己的強化金身及強化偵查技能，強化後的金身技能可以反彈一切受到的攻擊並將攻擊效果回饋給對手，包含普通攻擊、束縛、狙殺或其他絕對防護反彈的傷害。絕對防護擁有非常強大的感知能力，強化後的偵查即便處於隱身狀態的敵人也無法躲過絕對防護的感知。
